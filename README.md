# 🚕 Taxi Bot

Бот-диспетчер для службы такси в Telegram.

## Переменные окружения (Railway)

| Переменная | Значение |
|---|---|
| BOT_TOKEN | Токен от BotFather |
| DRIVERS_CHAT_ID | ID группы водителей |

## Как работает

1. Клиент пишет боту /start
2. Вводит откуда и куда
3. Заказ появляется в чате водителей
4. Водитель нажимает "Взять заказ"
5. Клиент и водитель общаются через бота анонимно
6. Водитель нажимает "Завершить заказ"
